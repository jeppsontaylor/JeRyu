# Security Policy

## Supported Versions

Currently, only the latest `main` branch of jeryu is supported with security updates. 

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities. Instead, reach out privately. 

Once your issue has been reported, we will:
- Acknowledge receipt within 48 hours.
- Investigate and provide a timeline for a patch.
- Issue a release with the security patch.
